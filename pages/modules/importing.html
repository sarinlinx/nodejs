Create and Import a module

Create the module

The replaceTemplate function could be added as a module so you can use it in multiple files.

In Node.js, all files are treated as a module.

Create a new file named replaceTemplate. Place the function code within it.

To export it, add module.exports where the variable name used to be.

module.exports = (temp, product) => {
let output = temp.replace(/{%PRODUCTNAME%}/g, product.productName);
output = output.replace(/{%IMAGE%}/g, product.image);
output = output.replace(/{%PRICE%}/g, product.price);
output = output.replace(/{%FROM%}/g, product.from);
output = output.replace(/{%NUTRIENTS%}/g, product.nutrients);
output = output.replace(/{%QUANTITY%}/g, product.quantity);
output = output.replace(/{%DESCRIPTION%}/g, product.description);
output = output.replace(/{%ID%}/g, product.id);

if(!product.organic) output = output.replace(/{%NOT_ORGANIC%}/g, 'not-organic');
return output;}

Import the module

Add a require at the top of your index.js file.

const replaceTemplate = require('./modules/replaceTemplate');

Import a module you installed

You just need to require it like any other Module. Node.js will then locate it in the node_modules directory.

const sluify = require('slugify')
